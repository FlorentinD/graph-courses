= Expanding Sandbox
:order: 1
:sandbox: true
:cypher: MATCH (n) RETURN count(n)


You can expand Sandbox by default by setting the `:sandbox:` attribute to `true`.


[source,adoc]
----
:sandbox: true
----


You can also pre-fill a cypher query in the window by setting the `:cypher:` attribute.


[source,adoc]
----
:cypher: MATCH (n) RETURN count(n)
----
