[TIP,role=hint]
.Hint
====
データベース内のパターンと一致するものを見つけるには、次のようなCypher文を実行します。

[source,cypher]
----
MATCH (p:Person) WHERE p.name = 'Daniel Kaluuya'
MATCH (m:Movie {title: 'Get Out'})
MATCH (p)-[:ACTED_IN]->(m)
RETURN p, m
----

**ヒント:** 2行目と3行目の最初のキーワードを変更するだけで、ムービーとリレーションシップが作成されます。
