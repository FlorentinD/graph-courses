[TIP,role=hint]
.Hint
====
データベース内のノードを見つけるには、次のようなCypher文を実行します。

[source,cypher]
----
MATCH (p:Person {name: 'Daniel Kaluuya'})
RETURN p
----
**Hint:** 必要なのは、クエリの最初のキーワードを変更することだけです。
====
