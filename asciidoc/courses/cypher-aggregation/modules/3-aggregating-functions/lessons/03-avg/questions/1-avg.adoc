[.question]
= 1. Elements used for `avg()`

Given this list:

----
[0,null,1,null,3,null,6,null,0,null]
----

What is the result of calling `avg()` on its elements?


* [ ] 1.0
* [x] 2.0

[TIP,role=hint]
.Hint
====
Null elements are not used in the calculation.
====

[TIP,role=solution]
.Solution
====
The correct answer os 2.0.
====
