= 1. First element of a list

What Cypher function returns the first element of a list?


* [ ] `first()`
* [x] `head()`
* [ ] `top()`


[TIP,role=hint]
.Hint
====
Think of going to the head of the line!
====

[TIP,role=solution]
.Solution
====
You use the `head()` function to return the first element of a list.
====
