= Challenge: Updating Properties
:order: 6
:usecase: movies
:sandbox: true
:verify: MATCH (m:Movie {title: 'Batman Begins'}) WHERE  m.tagline = 'This Summer, Evil Fears The Knight.' AND m.released = 2005 RETURN true AS outcome

We discovered that we created the _Movie_ node for the movie _Batman Begins_, but we forgot to add the tagline which is [copy]#This Summer, Evil Fears The Knight.# and the released year of 2005.

Write the Cypher code to find this _Movie_ node and add the properties to the node.

[TIP]
Make sure the _Movie_ node exists for _Batman Begins_
Use MATCH to find the _Movie_ node and then use SET to add the properties.
