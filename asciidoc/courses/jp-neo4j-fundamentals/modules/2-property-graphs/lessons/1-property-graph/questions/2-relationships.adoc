= 2. Neo4jでのリレーションシップ

Neo4jのリレーションシップは以下のうち何を必須としているでしょうか。

 * [ ] a property
 * [ ] a weight
 * [x] a type
 * [x] a direction

[TIP,role=hint]
.ヒント
====
Neo4jのすべてのリレーションは、この2つを備えていなければなりません。
====

[TIP,role=solution]
.解答
====
リレーションシップは、**direction（方向）** および **type（タイプ）** を持つ必要があります。
====
