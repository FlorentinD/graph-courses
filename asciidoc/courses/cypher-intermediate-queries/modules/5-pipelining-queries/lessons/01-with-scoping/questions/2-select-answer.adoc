[.question]
= 2. Cypher scoping

What Cypher keyword is used to redefine the scope of variables in a query?

* [ ] MATCH
* [ ] RETURN
* [x] WITH
* [ ] SCOPE

[TIP,role=hint]
.Hint
====
This clause can be used anywhere in a query to redefine scope. It is not `SCOPE`.
====


[TIP,role=solution]
.Solution
====
The correct answer is: `WITH`
====
