= 2. List inclusion

What Cypher keyword is used to test if a value is in a list property?

* [ ] ELEMENT OF
* [ ] INCLUDES
* [x] IN
* [ ] IS CONTAINED IN

[TIP,role=hint]
.Hint
====
In Cypher, you can test if a value is in a list using a single keyword.
====

[TIP,role=solution]
.Solution
====
`IN` is the correct answer.
====
