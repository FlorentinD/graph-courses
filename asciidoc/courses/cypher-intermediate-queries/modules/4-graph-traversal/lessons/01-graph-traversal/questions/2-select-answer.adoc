[.question]
= 2. Traversing the graph

What term best describes the traversal behavior during a query?

* [x] depth-first
* [ ] breadth-first


[TIP,role=hint]
.Hint
====
The graph engine always traverses a complete path through the graph.
====

[TIP,role=solution]
.Solution
====
The correct answer is: depth-first
====
