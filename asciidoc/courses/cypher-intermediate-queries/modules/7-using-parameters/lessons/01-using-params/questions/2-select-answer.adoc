= 2. Parameters for the session?

What command in Neo4j Browser returns all parameters set for the session?

* [ ] `:parameters`
* [ ] `:show parameters`
* [x] `:params`
* [ ] `:show params`

[TIP,role=hint]
.Hint
====
It is a very short one-word command.
====

[TIP,role=solution]
.Solution
====
The correct answer is: `:params`.
====
