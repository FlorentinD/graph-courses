[#{id}.question]
= 1. なぜラベルを追加するのか？

ノードにラベルを追加する主な理由は何ですか？

* [ ] 階層をモデル化するため
* [ ] データモデルを見やすくするため
* [x] 実行時にアクセスするデータ数を減らすため
* [ ] Cypherのコードを柔軟にし、パラメーターをつけやすくするため

[TIP,role=hint]
.ヒント
====
新しいラベルを追加することで、クエリ中に**アクセスするノードの数を減らし**たり、結果を特定のサブセットのノードにフィルタリングしたりすることができます。
====

[TIP,role=solution]
.解答
====
正解は、**実行時にアクセスするデータ数を減らすため**です。
====
